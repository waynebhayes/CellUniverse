# Embryo initial files

This folder organizes the generated embryo initial CSV files. The CSV files were moved only; their contents were not edited.

Categories:
- 00_core_start_points: broad manual or resume entry points such as frame 0, 53, 79, 84, 85, and 86.
- 01_early_verified_starts: early frame test starts around frames 15, 25, 35, 40, and 42.
- 02_gt_and_manual_late_starts: GT based or manual GT delay fixed starts.
- 03_fusion_chain_122_171: sequential CellLumen fusion starts generated from previous fusion frames.

See INITIAL_FILES_INDEX.tsv for the exact current path of every moved file.
